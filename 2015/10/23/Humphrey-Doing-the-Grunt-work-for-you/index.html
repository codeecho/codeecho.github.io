Humphrey - Doing the Grunt work for you

Humphrey is a command line utility which simplifies and speeds up using Grunt. It allows you to scaffold snippets of the grunt file and add them with a single command. To maximise the resuse of these snippets Humphrey adds a bit of convention to the grunt file with a standard directory structure and lifecycle

For example, the following code will setup a project which will run jshint across your javascript files, process Jade templates, minify your html and serve your app on a local port, while watching and rebuilding on any changes.
